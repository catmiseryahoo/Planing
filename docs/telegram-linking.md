# Telegram-уведомления в Orbita

Документация разделена по ролям.

- [Пользователь: привязка личного Telegram](telegram-user.md)
- [Администратор организации: Telegram-уведомления и канал организации](telegram-organization-admin.md)
- [Супер-администратор: Supabase, secrets и webhook](telegram-super-admin.md)

Коротко:

- Один общий бот приложения нужен всегда: через него сотрудники привязывают личный Telegram и получают личные уведомления.
- Канал не обязателен: он нужен только если организация хочет получать копию уведомлений в общий Telegram-канал.
- Администратор организации не настраивает бота и не работает с Supabase, базой, secrets и webhook. Это зона супер-администратора или технического администратора.
