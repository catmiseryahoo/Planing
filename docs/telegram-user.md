# Пользователь: привязка личного Telegram

1. Откройте Orbita.
2. Перейдите в личный кабинет.
3. В блоке Telegram нажмите `Получить код`.
4. Нажмите ссылку `Открыть корпоративного бота`.
5. Отправьте боту команду из личного кабинета в формате `/start 123456`.
6. Дождитесь ответа бота: `Telegram привязан к Orbita`.

Код действует 15 минут. Если срок истёк, получите новый код в личном кабинете.

После привязки уведомления будут приходить в личный Telegram, если:

- администратор организации включил Telegram в настройках организации;
- в вашей карточке сотрудника включён канал `Telegram`;
- сообщение адресовано вам или проектному чату, в котором вы участвуете.

Важно: бот не может первым написать человеку. Сначала пользователь сам открывает бота и отправляет `/start код`.
